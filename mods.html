WARNING
Respawn seems to now be enforcing bans for anyone known to be using mods they deem "unnacceptable". It's currently not 100% clear what they deem acceptable or not, but I've marked confirmed examples. FairFight cannot automatically ban you if you are using these, because it a completely server sided stat analysis tool. However, if you have public videos of yourself using these mods, you are at risk and should hide/delete them or risk incrimination. If you want to be completely safe of any potential trouble, I do recommend deleting the mods entirely so you just don't have to worry about this at all. In all cases, download these at your own risk.

Titanfall 2 mods

If you're just looking for a drag-and-drop installation, click the "Download Compiled VPK" link for a pre-compiled VPK, then just put the contents of the .zip into Origin Games\Titanfall2\vpk\. If you want to use several of these alongside one another, take note of which VPK it modifies; if they modify the same VPK, scroll down to the combined VPKs and use those.
If you're having issues with these compiled VPKs or just want to combine the mod with your own personal compile, click the "Download Source Files" link for a .zip that is only the raw modified files and then just dump those into the root directory of the appropriate VPK. If you're not up to speed on how to modify a VPK, I have a tutorial video here.

Japanese Localization for English Copies
Forces your game to use the Japanese localization when set to English, since it's otherwise not available through Origin without purchasing a Japanese copy of the game. Will change the UI language and all voice acting.
Be aware that for some reason you will not be able to see network invites with this active. Note that you will be unable to join invites from Networks whose regions don't include Japan.

Status: Safe
Does not modify any VPKs

Download (403.6 MB)

English Text & Subtitles for Japanese Localization
Companion mod to the above, will restore English UI text and subtitles while using the Japanese localization of the game. Not 100% perfect, server-side text like gamemodes and the FAQ will remain in Japanese. Won't help with the network invites either.

Status: Safe
Modifies VPK: englishclient_frontend

Download Compiled VPK (50.8 MB)
Download Source Files (430 KB)

Pilot Speedometer
In the Live Fire patch, Respawn snuck in WIP code for a Pilot speedometer, complete with imperial and metric displays. This mod will turn that on.
Will default to imperial, you can swap to metric by setting the launch convar +password metric in Origin.

Status: Safe
Modifies VPK: englishclient_mp_common

Download Compiled VPK (411.6 MB)
Download Source Files (4 KB)
