Titanfall 2 mods

If you're just looking for a drag-and-drop installation, click the "Download Compiled VPK" link for a pre-compiled VPK, then just put the contents of the .zip into Origin Games\Titanfall2\vpk\. If you want to use several of these alongside one another, take note of which VPK it modifies; if they modify the same VPK, you'll need to compile it manually with the sources.
If you're having issues with these compiled VPKs or just want to combine the mod with your own personal compile, click the "Download Source Files" link for a .zip that is only the raw modified files and then just dump those into the root directory of the appropriate VPK. If you're not up to speed on how to modify a VPK, I have a tutorial video here.

Japanese Localization for English Copies
Forces your game to use the Japanese localization when set to English, since it's otherwise not available through Origin without purchasing a Japanese copy of the game. Will change the UI language and all voice acting.
Be aware that for some reason you will be unable to join invites from Networks whose regions don't include Japan.

Does not modify any VPKs
Last Updated: v2.0.7.0
Download (534.4 MB)

English Text & Subtitles for Japanese Localization
Companion mod to the above, will restore English UI text and subtitles while using the Japanese localization of the game. Not 100% perfect, server-side text like gamemodes and the FAQ will remain in Japanese. Won't help with the network invites either.

Pilot Speedometer
In the Live Fire patch, Respawn snuck in WIP code for a Pilot speedometer, complete with imperial and metric displays. This mod will turn that on.
Will default to imperial, you can swap to metric by setting the launch convar +password metric in Origin.

No HUD
Made with help from Shig/ChristmasLights. He made this great video called "The Art of the Grapple", you should check it out! Removes almost all of the HUD, for a "hardcore" experience or something. To toot my own horn: probably better than Respawn's own no HUD command which hides things like holographic sights, while this won't.

Photo Mode
The No HUD mod, but this will also pull your viewmodel waaay back so that it's off-screen. Perfect for photos. This is an imperfect solution, as bigger weapons may still clip on screen sometimes when moving a lot and you'll still see your hands when getting on a zipline or ledge, but it's the best I can do without anything crashing for now.

Deprecated Mods

Each of these mods was made in response to issues that Respawn has by now addressed (even if not very completely, but at least adequately). As such, I don't feel any need to keep them updated since I don't even use them myself anymore. I'm keeping them on here for anyone who wants them to use anyways or to use as references.

Improved Crosshairs
This will give most Pilot weapons a plus and dot styled crosshair and adds dots to all Titan weapons. Straightforward.
Note that Respawn has gone on record that they don't approve of this mod in particular, so use at your own risk.

Improved Sunflares
